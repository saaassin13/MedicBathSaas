# 当班数据统计页面设计文档

**日期**：2026-04-23
**页面**：当班数据统计（数据统计分析）
**状态**：已批准
**关联页面**：故障日志信息、历史数据统计

---

## 1. 概述

### 1.1 目的
为"奶牛药浴大数据管理中心"提供当班数据统计分析功能，展示核心KPI指标、趋势图表、设备状态和维保状态。

### 1.2 范围
- KPI指标卡片
- 趋势图表（牛数/识别率、药液用量/均量）
- 环形图（识别率与覆盖率分析）
- 详细数据表格
- 设备状态监控
- 维保状态监控
- 独立日期筛选功能

### 1.3 设计依据
基于设计图 `demo/ui/数据统计信息-当班数据统计.png` 进行实现。

## 5. 功能需求

### 5.1 KPI指标卡片

| 指标名 | 数值示例 | 图标 | 说明 |
|--------|---------|------|------|
| 当班药浴牛数 | 425头 | 橙色图标 | 当班总计药浴的奶牛数量 |
| 当班乳头识别率 | 98.64% | 眼睛图标 | 视觉算法准确度 |
| 当班药液用量 | 145.15 L | 液桶图标 | 当班药液总消耗 |
| 当班药液均量 | 10.97 ML/头 | 水滴图标 | 平均每头奶牛药液用量 |

### 5.2 趋势图表

#### 药浴牛数/识别率统计分析

| 属性 | 值 |
|------|------|

| 横轴 | 日期（4/01 至 4/29） |
| 左轴 | 牛数（0-10k） |
| 右轴 | 识别率（0-100%） |
| 柱状颜色 | #1890FF（蓝色） |
| 折线颜色 | #FF8C00（橙色） |
| 筛选器 | 日期范围选择器 |

#### 药液用量/均量统计分析

| 属性 | 值 |
|------|------|
| 图表类型 | 双轴图表：柱状图（用量）+ 折线图（均量） |
| 横轴 | 日期（4/01 至 4/29） |
| 左轴 | 总用量（0-300L） |
| 右轴 | 均量（0-20ML） |
| 柱状颜色 | #52C41A（绿色） |
| 折线颜色 | #722ED1（紫色） |
| 筛选器 | 日期范围选择器 |

### 5.3 环形图（识别率与覆盖率分析）

| 属性 | 值 |
|------|------|
| 图表类型 | 环形图（Doughnut） |
| 中心数字 | 总数（140） |
| 数据分类 | 未脱杯、防褪链、牛腿过窄、异常跳过 |
| 分类颜色 | 待定（使用语义化配色） |
| 筛选器 | 日期范围选择器 |

**环形图数据结构：**

```typescript
interface DonutData {
  total: number;        // 总数：140
  categories: {
    name: string;        // 分类名称
    count: number;       // 数量
    percentage: number;  // 占比
  }[];
}

// 示例数据
const donutData: DonutData = {
  total: 140,
  categories: [
    { name: '未脱杯', count: 50, percentage: '35%' },
    { name: '防褪链', count: 40, percentage: '26%' },
    { name: '牛腿过窄', count: 20, percentage: '15%' },
    { name: '异常跳过', count: 15, percentage: '10%' }
  ]
};
```

### 5.4 详细数据表格

| 列名 | 说明 |
|------|------|
| 开始时间 | 作业开始时间 |
| 结束时间 | 作业结束时间 |
| 运行时长 | 持续时间 |
| 识别牛数 | 当班识别牛只数 |
| 识别乳头 | 识别乳头总数 |
| 未识别乳头 | 未识别乳头数 |

**表格筛选器**：日期范围选择器

interface DateFilterProps {

interface StatisticsState {

## 8. 技术选型

与前面页面保持一致：

| 技术 | 选型 | 说明 |
|------|------|------|
| 框架 | React 18 + TypeScript | 组件复用性强 |
| 样式 | TailwindCSS + CSS Module | 快速实现布局 |
| UI库 | Ant Design 5 | 企业级 B 端设计 |
| 路由 | React Router v6 | React 生态标准 |
| 图表库 | ECharts 或 Recharts | 数据可视化 |

### 8.1 图表库选型建议

| 库 | 优点 | 适用场景 |
|------|------|---------|
| ECharts | 功能强大，配置灵活，支持双轴图表 | 趋势图表、环形图 |
| Recharts | React原生，轻量易用 | 简单图表 |

**推荐使用 ECharts**，支持复杂双轴图表和环形图。
