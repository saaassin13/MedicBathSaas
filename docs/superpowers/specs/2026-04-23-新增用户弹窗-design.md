## 1. 概述

### 1.1 目的
为用户管理模块提供新增用户和编辑用户的功能弹窗，支持基本信息录入和权限分配。

### 1.2 范围
- 新增用户表单
- 编辑用户表单（复用）
- 权限树选择组件
- 表单验证

### 1.3 设计依据
基于设计图 `demo/ui/新增用户.png` 进行实现。

## 5. 表单字段

### 5.1 字段定义

| 字段 | 类型 | 必填 | 验证规则 | 提示文字 |
|------|------|------|---------|---------|
| 用户名 | text | 是 | 3-20字符，唯一性校验 | 请输入用户名，3-20字符 |
| 姓名 | text | 是 | 2-20字符 | 请输入姓名 |
| 电话 | text | 是 | 11位手机号 | 请输入电话 |
| 邮箱 | text | 是 | 邮箱格式 | 请输入邮箱 |
| 密码 | password | 新增时是 | 6位以上，字母或特殊符号和数字结合 | 请输入密码 |
| 状态 | switch | 否 | 默认启用 | - |

### 5.2 编辑模式差异

| 字段 | 新增模式 | 编辑模式 |
|------|---------|---------|
| 用户名 | 可编辑 | 可编辑（但不可修改） |
| 密码 | 必填 | 留空（不显示） |
| 权限 | 必选 | 可选 |

### 6.1 层级定义

```
权限树

│   ├── 东北大区

│       └── ...
```

### 6.2 树节点结构

```typescript
interface TreeNode {
  key: string;
  title: string;
  children?: TreeNode[];
  disabled?: boolean;
}
```

### 6.3 联动规则

| 操作 | 行为 |
|------|------|
| 勾选父节点 | 自动勾选所有子节点 |
| 取消勾选父节点 | 自动取消所有子节点 |
| 子节点全选 | 父节点自动变为勾选状态 |
| 子节点部分选中 | 父节点变为半选状态（indeterminate） |

## 7. 交互逻辑

### 7.1 弹窗打开/关闭

| 触发方式 | 行为 |
|---------|------|
| 点击"新增用户" | 打开弹窗，mode='create'，表单为空 |
| 点击"编辑" | 打开弹窗，mode='edit'，表单预填充 |
| 点击"X"按钮 | 关闭弹窗，清空表单 |
| 点击"取消"按钮 | 关闭弹窗，清空表单 |
| 点击遮罩区域 | 关闭弹窗（需二次确认是否有未保存数据） |

### 7.2 表单提交

```
点击"提交"按钮
    ↓
验证所有必填字段
    ↓
验证失败 → 显示错误提示，聚焦到第一个错误字段
    ↓
验证通过 → 调用 API（新增/更新）
    ↓
API 成功 → 关闭弹窗，刷新用户列表
API 失败 → 显示错误提示
```

### 7.3 权限搜索

```
输入搜索关键词
    ↓
防抖 300ms
    ↓
过滤权限树节点（模糊匹配 title）
    ↓
高亮匹配文字
    ↓
无匹配结果 → 显示"无匹配结果"
```

## 10. 关联说明

### 10.1 与用户管理列表页联动

| 触发 | 弹窗行为 |
|------|---------|
| 用户管理 → 点击"新增用户" | 打开新增弹窗（空白表单） |
| 用户管理 → 点击"编辑" | 打开编辑弹窗（预填充数据） |
| 弹窗 → 提交成功 | 关闭弹窗，用户管理列表刷新 |
| 弹窗 → 关闭 | 保持用户管理列表当前状态 |

### 10.2 共享组件

用户管理列表页和新增用户弹窗共享以下组件：
- UserPermissionTree - 权限树组件
- FormInput - 表单输入组件（后续通用化）
