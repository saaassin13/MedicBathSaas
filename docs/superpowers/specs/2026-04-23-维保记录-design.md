## 1. 概述

### 1.1 目的
为"奶牛药浴大数据管理中心"提供维保历史记录的查询功能，支持多维度筛选和数据追溯。

### 1.2 范围
- 维保记录列表展示
- 奶厅筛选
- 维保内容筛选
- 时间范围筛选

### 1.3 设计依据
基于设计图 `demo/ui/维保记录.png` 进行实现。

## 5. 功能需求

### 5.1 筛选功能

| 筛选器 | 类型 | 默认值 | 选项 |
|--------|------|--------|------|
| 奶厅选择 | 下拉 | 1号奶厅 | 全部、1号奶厅、2号奶厅... |
| 维保内容 | 下拉 | 全部维保内容 | 全部、更换编码器电池、喷枪螺栓紧固... |
| 开始日期 | 日期选择 | 空 | - |
| 结束日期 | 日期选择 | 空 | - |

### 5.2 筛选联动逻辑

```
用户设置筛选条件
    ↓
点击"搜索"或自动触发（根据设计待确认）
    ↓
调用API获取过滤后的记录
    ↓
更新列表显示
    ↓
无结果 → 显示"暂无记录"
```

### 5.3 日期范围验证

- 结束日期不能早于开始日期
- 默认显示最近30天的记录

## 6. 日志列表结构

### 6.1 记录格式

| 字段 | 内容 | 格式说明 |
|------|------|---------|
| 时间戳 | 精确到毫秒的时间 | YYYY/MM/DD HH:mm:ss.SSS |
| 状态 | "完成" | 固定值，表示已执行 |
| 任务详情 | 具体维保动作 | 如：更换机器人编码器电池 |
| 分类 | "维保任务" | 固定标签，区分其他类型 |

### 6.2 时间戳格式

```typescript
// 格式：YYYY/MM/DD HH:mm:ss.SSS
// 示例：2026/03/07 00:00:00.000

interface MaintenanceRecord {
  id: string;
  timestamp: string;        // 格式：YYYY/MM/DD HH:mm:ss.SSS
  status: 'completed';     // 目前只有完成状态
  content: string;          // 维保内容描述
  category: '维保任务';    // 固定值
  hallId: string;          // 关联的奶厅ID
}
```

## 7. 与维保计划页面的关系

### 7.1 页面职责分工

| 页面 | 职责 | 时机 |
|------|------|------|
| 维保计划与任务 | 事前管理，提醒未来维保任务 | 将来时 |
| 维保记录 | 事后留痕，记录已完成的维保 | 过去时 |

### 7.2 数据流转

```
维保计划 → 执行维保 → 生成维保记录
    ↑                           ↓
    ← 维保记录页面可查看历史 ←
```
