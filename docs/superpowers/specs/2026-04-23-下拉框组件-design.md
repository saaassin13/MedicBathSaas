## 1. 概述

### 1.1 目的
为"奶牛药浴大数据管理中心"提供通用的级联下拉选择器组件，支持复杂的层级数据筛选。

### 1.2 范围
- 五级级联选择器组件
- 单选模式
- 多选模式
- 搜索过滤功能
- 路径高亮显示

### 1.3 设计依据
基于设计图 `demo/ui/下拉框.png` 进行实现。

## 4. 组件 Props

### 4.1 基本属性

```typescript
interface CascaderProps {
  /** 层级数据源 */
  options: CascaderOption[];

  /** 默认值（受控模式） */
  value?: string[];

  /** 默认值（非受控模式） */
  defaultValue?: string[];

  /** 选择模式：单选或多选 */
  mode: 'single' | 'multiple';

  /** 占位符 */
  placeholder?: string;

  /** 是否显示搜索框 */
  showSearch?: boolean;

  /** 选择完成后的回调 */
  onChange?: (value: string[], selectedOptions: CascaderOption[]) => void;

  /** 是否禁用 */
  disabled?: boolean;

  /** 组件宽度 */
  width?: number | string;
}

interface CascaderOption {
  value: string;
  label: string;
  children?: CascaderOption[];
  disabled?: boolean;
}
```

### 4.2 单选模式返回值

```typescript
// 单选模式：返回选中末级的完整路径
onChange: (value: string[]) => void;
// 示例：['region-northeast', 'farm-shuangcheng', 'hall-1', 'rotary-1', 'process-front']
```

### 4.3 多选模式返回值

```typescript
// 多选模式：返回选中末级的路径数组
onChange: (value: string[]) => void;
// 示例：['.../前药浴', '.../后药浴']
```

## 5. 交互逻辑

### 5.1 展开/收起

| 交互 | 行为 |
|------|------|
| 点击输入框 | 展开级联面板 |
| 点击文档空白处 | 收起面板 |
| 按 ESC 键 | 收起面板 |
| 点击输入框右侧箭头 | 切换展开/收起 |

### 5.2 层级展开

| 交互 | 行为 |
|------|------|
| 悬停/点击有子级的行 | 右侧显示下一级面板 |
| 点击无子级的行（末级） | 选中并关闭（单选模式） |
| 逐级展开 | 保持当前路径高亮 |

### 5.3 单选模式

```
点击"前药浴"
    ↓
选中该选项
    ↓
输入框显示："东北大区 / 双城牧场 / 1期奶厅 / 1号转盘 / 前药浴"
    ↓
关闭面板
```

### 5.4 多选模式

```
勾选"前药浴"
    ↓
继续保持面板打开
    ↓
勾选"后药浴"
    ↓
输入框显示："已选 2 项"
    ↓
点击空白处或确认按钮关闭面板
```

### 5.5 路径高亮

当选择某个末级后，从第一级到该末级的完整路径都应该高亮显示（橙色文字），方便用户了解选择的位置。

## 7. 组件结构

```
src/
├── components/
│   └── common/
│       ├── Cascader.tsx            # 主组件
│       ├── CascaderPanel.tsx       # 级联面板
│       ├── CascaderOption.tsx      # 单行选项
```

### 7.1 组件职责

| 组件 | 职责 |
|------|------|
| Cascader | 主组件，管理展开/收起、模式切换 |
| CascaderPanel | 级联面板容器，渲染多列 |

| CascaderSearch | 搜索框组件，防抖过滤 |

| MultipleMode | 多选模式逻辑 |

## 8. 技术选型

| 技术 | 选型 | 说明 |
|------|------|------|
| 框架 | React 18 + TypeScript | 组件复用性强 |
| 样式 | TailwindCSS + CSS Module | 快速实现布局 |
| 现有库 | Ant Design Cascader | 可基于Antd二次开发，也可自研 |

### 8.1 现有方案对比

| 方案 | 优点 | 缺点 |
|------|------|------|
| 基于 Ant Design Cascader | 功能完善，稳定 | 定制化受限，需覆盖样式 |
| 自研组件 | 完全可控，定制灵活 | 开发周期长 |

**推荐**：基于 Ant Design Cascader 二次开发，覆盖品牌色样式。
