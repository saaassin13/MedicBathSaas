## 1. 概述

### 1.1 目的
为"奶牛药浴大数据管理中心"构建奶厅作业实时监控页面，展示12路监控视频的状态概览。

### 1.2 范围
- 监控网格布局
- 监控卡片组件（含状态标签、位置信息）
- 奶厅筛选功能
- 图片占位符（后续接入实时监控）

### 1.3 设计依据
基于设计图 `demo/ui/奶厅作业监控.png` 进行实现。

## 5. 功能需求

### 5.1 奶厅筛选

| 功能 | 描述 |
|------|------|
| 筛选下拉 | 显示所有奶厅选项 |
| 选项内容 | 全部奶厅、东北大区-双城牧场-1期奶厅、... |
| 选择后 | 更新监控卡片列表（筛选显示的奶厅） |

### 5.2 监控卡片

每个卡片包含以下区域：

```
┌─────────────────────────────────┐
│ ● 运行正常                      │ ← 状态标签（左上角）
│                                 │
│        [监控画面区域]            │ ← 占位图片或视频
│         (16:9 比例)              │
│                                 │
├─────────────────────────────────┤
│ 蒙牛集团-东北大区-双城-1期奶厅  │ ← 位置信息浮层
│ -1号转盘-后药浴-监控视角          │
└─────────────────────────────────┘
```

### 5.3 监控卡片数据结构

```typescript
interface MonitorCard {
  id: string;
  location: {
    group: string;      // 蒙牛集团
    region: string;      // 东北大区
    farm: string;       // 双城
    milkingHall: string; // 1期奶厅
    equipment: string;  // 1号转盘
    process: string;    // 前药浴/后药浴
    viewType: string;   // 监控视角/作业视角
  };
  status: 'normal' | 'error' | 'stopped';
  thumbnail: string;    // 当前为占位图片，后续为实时视频
}
```

### 5.4 占位图片说明

**当前阶段**：使用静态占位图片
- 每个监控卡片显示一张占位图片
- 占位图片包含简单的牧场/设备示意

**后续阶段**：接入实时监控
- 占位图片替换为实时视频流
- 使用 WebRTC 或 RTSP 转 WebSocket 技术
- 视频播放器使用 hls.js 或 video.js

    ↓
筛选符合条件的监控卡片
