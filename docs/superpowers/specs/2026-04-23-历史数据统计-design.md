## 1. 概述

### 1.1 目的
为"奶牛药浴大数据管理中心"提供历史作业数据的查询、追溯和导出功能，支持多维度筛选和分页浏览。

### 1.2 范围
- 历史数据表格展示
- 位置筛选
- 日期范围筛选
- 三级导出功能（单条/批量/全量）
- 分页组件

### 1.3 设计依据
基于设计图 `demo/ui/数据统计信息-历史数据统计.png` 进行实现。

## 5. 功能需求

### 5.1 筛选功能

| 筛选器 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| 位置筛选 | 下拉 | 全部 | 按奶厅/区域筛选 |
| 开始日期 | 日期选择 | 空 | 历史记录起始时间 |
| 结束日期 | 日期选择 | 空 | 历史记录结束时间 |

### 5.2 数据表格列定义

| 列名 | 宽度 | 说明 |
|------|------|------|
| 勾选框 | 48px | 单选/全选 |
| 奶厅 | 200px | 完整路径（如：双城-1期奶厅-1号转盘-后药浴） |
| 开始时间 | 160px | 格式：YYYY-MM-DD HH:mm |
| 结束时间 | 160px | 格式：YYYY-MM-DD HH:mm |
| 运行时长 | 100px | 格式：HH:mm:ss |
| 识别牛数 | 80px | 数字 |
| 识别乳头 | 80px | 数字 |
| 未识别乳头 | 80px | 数字 |
| 乳头识别率 | 100px | 百分比，如：97.85% |
| 识别牛只 | 80px | 头数 |
| 未脱杯 | 80px | 个数 |
| 防褪链 | 80px | 个数 |
| 牛腿过窄 | 80px | 个数 |
| 异常跳过 | 80px | 个数 |
| 漏喷牛只 | 80px | 个数 |
| 操作 | 80px | 单行导出链接 |

### 5.3 导出功能

| 层级 | 操作 | 触发方式 | 按钮样式 |
|------|------|---------|---------|
| 单条导出 | 导出单行数据 | 点击行末"导出"链接 | 蓝色文字链接 |
| 批量导出 | 导出勾选的数据 | 勾选后点击"导出" | 绿色填充按钮 |
| 全量导出 | 导出所有符合筛选条件的数据 | 点击"导出全部" | 蓝色填充按钮 |

**导出格式**：支持 Excel（.xlsx）、CSV

### 5.4 分页组件

| 功能 | 说明 |
|------|------|
| 每页条数 | 可选：10、20、50 |
| 当前页码 | 显示当前页，如：第 1 页 |
| 总页数 | 显示总页数，如：共 100 页 |
| 页码跳转 | 点击数字跳转，或输入页码跳转 |
| 上一页/下一页 | 翻页按钮 |

## 6. 数据结构

### 6.1 历史记录数据模型

```typescript
interface HistoryRecord {
  id: string;
  hall: string;              // 奶厅完整路径
  startTime: string;         // 开始时间 YYYY-MM-DD HH:mm
  endTime: string;           // 结束时间 YYYY-MM-DD HH:mm
  duration: string;          // 运行时长 HH:mm:ss
  recognizedCows: number;    // 识别牛数
  recognizedNipples: number; // 识别乳头
  unrecognizedNipples: number;// 未识别乳头
  recognitionRate: number;   // 乳头识别率百分比
  recognizedCowCount: number; // 识别牛只（头）
  notCupRemoved: number;      // 未脱杯（个）
  antiFadeChain: number;      // 防褪链（个）
  narrowCowLeg: number;       // 牛腿过窄（个）
  abnormalSkip: number;      // 异常跳过（个）
  missedSprayCows: number;   // 漏喷牛只（个）
}

interface HistoryState {
  records: HistoryRecord[];
  selectedIds: string[];
  filters: {
    hall: string;            // 筛选的奶厅
    startDate: string;       // 开始日期
    endDate: string;         // 结束日期
  };
  pagination: {
    page: number;
    pageSize: number;
    total: number;
  };
  loading: boolean;
}
```

## 7. 与当班数据统计页面的区别

| 特性 | 当班数据统计 | 历史数据统计 |
|------|-------------|-------------|
| 时间范围 | 当前班次/24小时内 | 跨月/跨年 |
| 数据状态 | 实时更新 | 已归档静态 |
| 功能重心 | 监控与即时处理 | 追溯、审计、导出 |
| KPI卡片 | 有（4个） | 无 |
| 图表 | 有（3个图表） | 无 |
| 表格列数 | 9列 | 16列（更详细） |
| 分页 | 无 | 有（支持大量数据） |
| 导出功能 | 无 | 三级导出 |
