## 1. 概述

### 1.1 目的
为"奶牛药浴大数据管理中心"提供设备故障和系统异常的日志查询功能，支持多维度筛选和故障追溯。

### 1.2 范围
- 故障日志列表展示
- 奶厅筛选
- 故障等级筛选（提示/警告/错误）
- 日期范围筛选

### 1.3 设计依据
基于设计图 `demo/ui/故障日志信息.png` 进行实现。

## 5. 功能需求

### 5.1 筛选功能

| 筛选器 | 类型 | 默认值 | 选项 |
|--------|------|--------|------|
| 奶厅选择 | 下拉 | 1号奶厅 | 全部、1号奶厅、2号奶厅... |
| 故障等级 | 下拉 | 全部故障等级 | 全部、提示、警告、错误 |
| 开始日期 | 日期选择 | 空 | - |
| 结束日期 | 日期选择 | 空 | - |

### 5.2 筛选联动逻辑

```
用户设置筛选条件
    ↓
触发搜索（或自动搜索）
    ↓
调用API获取过滤后的日志
    ↓
更新列表显示
    ↓
无结果 → 显示"暂无日志记录"
```

### 5.3 日期范围验证

- 结束日期不能早于开始日期
- 默认显示最近7天的记录

## 6. 日志列表结构

### 6.1 日志类型定义

```typescript
type LogLevel = 'info' | 'warning' | 'error';

interface FaultLog {
  id: string;
  timestamp: string;           // 格式：YYYY/MM/DD HH:mm:ss.SSS
  level: LogLevel;             // info/warning/error
  content: string;              // 日志内容描述
  hallId: string;               // 关联的奶厅ID
}

// 日志级别对应关系
const LOG_LEVEL_CONFIG = {
  'info': {
    label: '提示',
    color: '#1890FF',
    examples: ['药浴任务开始', '药浴任务结束']
  },
  'warning': {
    label: '警告',
    color: '#FAAD14',
    examples: ['机器人编码器电池已过期']
  },
  'error': {
    label: '错误',
    color: '#FF4D4F',
    examples: ['光源异常', '相机连接断开', '机器人连接断开']
  }
};
```

## 7. 与其他页面的关系

### 7.1 页面职责对比

| 页面 | 内容 | 时间戳 |
|------|------|--------|
| 维保记录 | 维保任务完成记录 | 精确到毫秒 |
| 故障日志 | 提示/警告/错误事件 | 精确到毫秒 |
| 维保计划与任务 | 未来维保任务提醒 | 无时间戳 |

### 7.2 故障处理流程

```
故障发生 → 记录到故障日志
    ↓
根据故障等级判断：
    ↓
错误/警告 → 触发维保计划创建
    ↓
维保执行 → 记录到维保记录
```
