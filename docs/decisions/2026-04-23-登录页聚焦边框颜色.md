# 决策：登录页输入框聚焦边框颜色统一为 #FF7300

**日期**：2026-04-23
**背景**：阶段三计划中 Login.module.css 的 `.input-field:focus` 使用了 #FF8C00，但登录页 spec 定义的主色调为 #FF7300
**选项**：
- A：保持 #FF8C00（与品牌色统一）
- B：改为 #FF7300（严格遵循 spec 定义）
**选择**：B，因为 rules/03-solution-code-consistency.md 要求代码必须与 spec 保持一致
**后果**：后续所有页面若涉及登录区域输入框聚焦色，应统一使用 #FF7300
